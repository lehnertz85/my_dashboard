Welcome to Plex Dashboard!

The top-right is a button to the menu. The menu lists all the services you wish to view. The cog next to the menu takes you to the settings. Settings is where you can modify the services in the menu. You can enable, disable, delete or add more.

  • Bear in mind, some websites don't allow themselves to be displayed in an iFrame. Github in one of them.

Icons:

Google Material icons are used. You can go to Google's materiial.io and search for icons. Once you have identified the icon you want, note the name. If the name includes multiple words, then type a _ between the words.

  • Example: "zoom in" would turn into "zoom_in".
